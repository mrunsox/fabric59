This is the documentation copy of the system prompt used by the `generate-script` edge function.
Update both the function and this file when changes are made.

PURPOSE
Generate a complete agent script (greeting, qualification questions, branching, closing) from a campaign description.

CONSTRAINTS
- Output strict JSON matching the script schema.
- Always include a fallback path for unqualified callers.
- Never include legal or medical advice in script copy.
